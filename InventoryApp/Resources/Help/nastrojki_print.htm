Настройки

Данные в настройках сохраняются автоматически при выходе Домой.
 

Резервное копирование базы данных

Состоит из пунктов: создать и восстановить из копии. Создает снимок текущей базы данных и сохраняется по выбранному пути. ВАЖНО! невозможно удалить базу данных или переместить ее пока в системе есть снимок, закрепленный за этой базой данных.
 

Прочие настройки

  • Показывать превью автоматически - позволяет показывать изображение выбранного товара сразу после выбора его в списке.
  • Сохранять документы автоматически - сохраняет документы поставок\отгрузок автоматически в соответсвующие папки, соритируя по дате. Доступны по нажатию Ctrl + P (0) или в меню.
  • Сброс пароля администратора, при следующем входе потребуется задать новый пароль администратора.
  • Вместимость склада - отображает максимальное физическое количество товара на складе.
 

Редактировать статусы

Отображает уже добавленные в систему статусы клиентов (название и размер скидки в процентах). Данные редактируются в соответсвующих полях. Для добавления нового статуса необходимо нажать на кнопку "Добавить статус".
 

Редактировать группы

Отображает уже добавленные в систему группы товаров (название группы и налог на группу товаров). Данные редактируются в соответсвующих полях. Для добавления новой группы необходимо нажать на кнопку "Добавить группу".
 

Редактировать типы магазинов

Отображает уже добавленные в систему типы магазинов (название типа). Данные редактируются в соответсвующих полях. Для добавления нового типа необходимо нажать на кнопку "Добавить тип".
