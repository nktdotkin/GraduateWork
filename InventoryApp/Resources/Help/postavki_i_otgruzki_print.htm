Поставки и отгрузки

 

Поставки

 
Для создания поставки на товар необходимо: иметь добавленного поставщика (см. раздел Поставщики) и хотя бы один товар. Размер поставки не должен выходить за физические размеры склада (см. раздел Настройки).
При создании поставки формируется отчет, который может быть использован для добавления товара. Нажмите Ctrl + P что бы открыть папку с отчетами поставок.
 

Отгрузки

 
Для создания отгрузки клиенту необходимо: иметь добавленного клиента (см. раздел Клиенты) и хотя бы один товар. Размер отгрузки должен быть более 0, если товара на складе не хватает, клиенту будет добавлено актуальное количество товара.
При создании отгрузки формируется отчет. Нажмите Ctrl + O что бы открыть папку с отчетами поставок.
